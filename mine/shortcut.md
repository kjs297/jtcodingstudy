## CMD
1. cd (change directory)  
  cd + \ : 루트 디렉토리로 이동  
  cd + \폴더\폴더(tab키 사용가능)  
  cd + .. : 빠꾸  
2. dir (폴더 내 검색)  

## Windows  
1. ctrl + Win + D : 가상데스크톱 만들기  
   ctrl + Win + 좌/우 : 가상데스크톱 이동  
   Win + tab : 삭제  
   Win + R : 실행  
   Win + 숫자 : 작업표시줄 단축키

## Atom  
1. Ctrl + , : 설정  
2. Ctrl + N : 새탭
3. Ctrl + shift + N : 새창
4. Ctrl + shift + M : 마크다운 프리뷰  
5. `<br>` : 라인브레이크 개행 한줄띄어쓰기
